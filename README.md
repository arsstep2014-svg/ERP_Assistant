# ERP AI Assistant

Мобильное приложение для управления задачами и сотрудниками предприятия, разработанное в рамках курсовой работы по исследованию архитектурного паттерна MVVM при создании клиент-серверных приложений на SwiftUI.

## Возможности

- просмотр списка сотрудников, полученного из внешнего REST API (MockAPI);
- локальное хранение задач с использованием SwiftData;
- экран KPI;
- AI-раздел со справочными ресурсами;
- профиль пользователя с настройками темы оформления.

## Используемые технологии

- Swift
- SwiftUI
- MVVM
- SwiftData
- URLSession
- MockAPI

## Требования

- Xcode 16 или новее
- iOS 18.0+

## Запуск проекта

1. Клонировать репозиторий:

```bash
git clone https://github.com/USERNAME/ERP-AI-Assistant.git
```

2. Открыть файл `ERP AI Assistant.xcodeproj` в Xcode.

3. Выбрать симулятор или подключенное устройство.

4. Нажать **Run (⌘R)**.

## Настройка API

Проект использует сервис MockAPI.

URL API задается в файле:

```
EmployeesAPIService.swift
```

При необходимости можно заменить адрес на собственный endpoint MockAPI.

## Дополнительная информация

Проект не использует сторонние библиотеки и не требует установки зависимостей, настройки CocoaPods, Swift Package Manager или API-ключей.
